# 프로잭트 개요
- JAVA Swing 을 활용한 은행 계좌 관리 시스템입니다.
- 사용자 인증 후 입금, 출금, 예금 조회 기능을 제공하며, 관리자 계정을 토한 시스테 ㅁ초기화 기능이 구현되어 있습니다.
- 사용자는 입출금 계좌와 예금 조회 계좌로 구분하였습니다.

- 언어 : JAVA
